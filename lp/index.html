よくある質問

FAQ

Q 具体的な実施の流れは？
ワーク単体の流れ：①お題の作成 → ②ワーク実施（1問あたり10〜15分、1名〜数百名まで）→ ③リアルタイムでアウトプット生成 → ④アウトプットの活用

プログラム全体の流れ：①目的・現状のヒアリング → ②体験会の実施 → ③プログラム・アウトプット設計の提案 → ④実施 → ⑤分析・レポート
Q 対象者の人数や役職は？
同時に1名から300名程度まで対応可能です。数万人規模の対象がいらっしゃる場合は複数グループに分けての実施になりますので、ご相談ください。
Q プログラムの期間や所要時間は？
1つのテーマに対して15分で作業完了します。オリエンテーションと3〜5ワークをセットして、1回のセッションが60〜90分が目安です。プログラム全体は3週間〜半年が一般的です。
Q 対応している言語は？
日本語、英語、インドネシア語、タイ語に対応しています。（近日中にベトナム語も対応予定）
Q 他社サービスとの違いは？
コーチング・1on1との違い：人によるサポートなしに本人の内発的な思考をアウトプットします。再現性が高く、コーチへの依存がありません。

インプット型研修との違い：インプット型の研修と組み合わせることで、インプットを消化・定着させることが可能です。

サーベイとの違い：定量的な物差しではなく、柔軟で多様な現場課題に即した各人の認識の生データを抽出します。
